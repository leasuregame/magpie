游戏 Web API

[ ]中得字符为参数。[platform]为平台参数（app或tb）去更新下载包
/api/[platform]/version
获取最新的版本号
/api/[platform]/update/[version]

获取更新包。[version]为当前手机游戏的版本号，根据这个版本号来判断是下载部分包还是完全包。

  • version 为空，下载完全包
  • version 为上一版本号，下载部分包
  • version 小于上一版本号，下载完全包
